Technical Projects

Home Lab Research

Summary: After completing Google’s Cybersecurity professional certification, I was inspired to continue my research into areas beyond the program’s scope, including network architecture, virtualization, web development and server management. As a result, I created a home testing lab to continue my technology-related learning and research. Below is a summary of these projects:

Personal Lab & Security Research Environment

Virtualization, Ubuntu Server and Kali Linux
  • Engineered a multi-VM lab to simulate enterprise environments for penetration testing, web development and network security research.
  • Deployed OWASP Juice Shop on a dedicated server to practice identifying vulnerabilities using Burp Suite application and Kali Linux tools installed on virtual machines.
  • Used virtual machines to simulate “red versus blue team” pen-testing and defense.
  • Repurposed legacy hardware by performing an operating system migration to Linux.
  • Converted a modern laptop to Linux-based Pop!_OS after reaching hardware limits, enabling additional virtual machine capacity.
  • Gained extensive experience with the command line interface (CLI) through R&D projects.
  • Researched and implemented RAM upgrades on multiple hardware units to improve lab capabilities.

Full-Stack Web Infrastructure (LEMP)

Linux, Nginx server, MySQL and PHP
  • Developed a custom portfolio website from scratch, utilizing GitHub Pages.
  • Architected and maintained a LEMP stack on multiple virtual Ubuntu servers to host dynamic, database-driven websites and applications.
  • Created and tested several MySQL databases in multiple virtual environments.
  • Integrated third-party REST APIs into several test websites.
  • Utilized AI-assisted development to optimize efficiency and troubleshoot complex scripts.
  • Published test projects and websites to multiple virtual servers.
  • Created and tested a mobile app using VS Code, Node.js and Android Studio.

Self-Hosted Learning Management System

Moodle, Database Management
  • Deployed and configured a Moodle LMS instance, managing the full lifecycle from server provisioning to administrative setup.
  • Created and published diverse content types, including full curricula, H5P, and SCORM-based courses.
  • Explored and tested various authoring tools to generate interactive content.

Reflections: I originally conceived of creating a home lab to provide hands-on experience with the concepts I’d been introduced to during the cybersecurity program. While this goal was accomplished, one of the most valuable aspects was overcoming unanticipated challenges.

I’ve learned additional ways to work with AI as a learning and problem-solving partner, especially when working through highly technical concepts that were initially unfamiliar.
